# Security Policy

## Supported Versions

Only the `master` branch receives security updates.
When an security update is made, it is released with the [latest version](../changelog).

| Version       | Supported          |
| ------------- | ------------------ |
| master        |       ✅           |
| anything else |       ❌           |

## Reporting a Vulnerability

If you find a security vulnerability,
please [report the vulnerability here](https://github.com/niccokunzmann/open-web-calendar/security/) or reach out to
[@niccokunzmann](https://github.com/niccokunzmann/).
